# Character Counter

A simple and responsive **Character Counter** web application built using **HTML, CSS, and JavaScript**. This project allows users to type text into a textarea and instantly displays the number of characters typed and the remaining characters based on a predefined limit.

## Features
- ✨ Real-time character counting
- 📏 Displays remaining character limit
- 📱 Responsive and user-friendly interface
- 🎨 Clean and modern design
- 🚀 Built using pure HTML, CSS, and JavaScript (no external libraries)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How It Works
- Type text into the input area.
- The application updates the character count instantly.
- The remaining character count decreases as you type.
- The maximum character limit is 200 characters.

## Purpose
This project was created to practice front-end web development concepts such as DOM manipulation, event handling, and responsive UI design using HTML, CSS, and JavaScript.
